## Quick Win: Apply This Today

**Set up a CLAUDE.md for your current project.** Run `/init` in your project directory to auto-generate one, then add:

```markdown
# Project Context
- This is a BCG consulting project for [client/topic]
- Data files are in /data, outputs go to /outputs
- Use pandas for data manipulation, matplotlib/seaborn for charts
- Always save plots as PNG files, never inline
- Use uv for package management

# Common Tasks
- Data cleaning: handle missing values, standardize column names
- Analysis: summary statistics, trend analysis, segmentation
- Output: client-ready charts with clear titles and labels
```

This gives Claude persistent context about your project, so every prompt starts smarter. Takes 2 minutes, saves hours over the course of a project.

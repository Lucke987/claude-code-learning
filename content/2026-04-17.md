## Today's Resources

### 1. Claude Code Routines: Automate Recurring Tasks in the Cloud (NEW - launched April 14)

**Time: ~5 min read**
**Link:** [Automate work with routines - Claude Code Docs](https://code.claude.com/docs/en/routines)

**Why it's relevant:** Routines let you package a prompt + repo + connectors into a saved configuration that runs automatically on Anthropic's cloud -- even when your laptop is closed. For a consultant juggling multiple workstreams, this means you can schedule tasks like "refresh the weekly data dashboard every Monday at 7am" or "review all open PRs on the client repo nightly" without lifting a finger.

**Key Takeaway:** Create your first routine via the CLI with `/schedule` (e.g., `/schedule daily data refresh at 8am`), from the Desktop app under "New remote task", or at [claude.ai/code/routines](https://claude.ai/code/routines). Routines can be triggered on a schedule, via API, or by GitHub events. Pro plans get 5 runs/day, Max gets 15, Team/Enterprise gets 25.

## Quick Win: Set Up Your First Custom Command Today

Create a reusable command for something you do every day. Here's a concrete example:

```bash
mkdir -p .claude/commands
cat > .claude/commands/analyze-data.md << 'EOF'
Analyze the data file at $ARGUMENTS:
1. Summarize the dataset (rows, columns, types, missing values)
2. Identify the top 3 most interesting patterns or anomalies
3. Suggest 2-3 visualizations that would best communicate findings to a client
4. Flag any data quality issues that need to be addressed
EOF
```

Then use it anytime with: `/project:analyze-data path/to/data.csv`

This takes 30 seconds to set up and saves minutes every time you start a new data analysis. Stack more commands for your other recurring tasks (deck review, document formatting, code cleanup) to build your personal consulting toolkit inside Claude Code.

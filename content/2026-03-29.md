## Quick Win

**Set up CLAUDE.md in your current project today.** Run `/init` in Claude Code to auto-generate it, then add:
- Your preferred Python libraries and data directories
- Common bash commands you use (e.g., `pytest`, `python run_analysis.py`)
- Project-specific conventions (naming, file structure, output formats)

This 2-minute setup means every future Claude Code session starts with full project context -- no more re-explaining your setup. From the BCG expert interviews, the teams seeing the biggest productivity gains are those who invest in making Claude hyper-contextual to their specific workflow.

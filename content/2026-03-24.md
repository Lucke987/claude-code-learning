## Quick Win — Apply This Today

**Build a `/bcg-slide` skill in 10 minutes.**

This is a one-time setup that pays off every time you need to translate analysis output into slide-ready content. Create the file at `~/.claude/skills/bcg-slide/SKILL.md`:

```markdown
---
name: bcg-slide
description: Format analysis output as BCG-style slide content
---

Take the analysis or data in this conversation and format it as slide-ready content:

1. **Slide title** — one action-oriented sentence (what the data proves, not what it shows)
2. **3 supporting bullets** — max 12 words each, quantified where possible, no passive voice
3. **So what** — one sentence: why this matters for the decision/recommendation
4. **Data source note** — where the numbers came from

Do not include narrative paragraphs. Output only the structured slide content above.
```

Save it. Now any time you've just finished an analysis and need to turn it into a slide, type `/bcg-slide` and Claude formats it immediately — no re-explaining the output style, no reformatting by hand.

Works in combination with yesterday's CLAUDE.md tip: keep your engagement context in `CLAUDE.md`, and your output format in a personal skill. Between the two, you've eliminated the two biggest sources of repetitive prompting.

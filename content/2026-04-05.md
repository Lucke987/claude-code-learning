## Resource 1: MCP Servers for Claude Code — Setup & Best Picks

**Source:** [Claude Code MCP Servers: How to Connect, Configure, and Use Them — Builder.io](https://www.builder.io/blog/claude-code-mcp-servers)
**Time:** ~5 min read
**Category:** Tool Integration

### Why This Is Relevant
MCP (Model Context Protocol) servers are how Claude Code connects to external tools — databases, GitHub, browsers, APIs — through a single open standard. For consulting work, this means Claude Code can directly query a PostgreSQL database with client data, manage PRs via the GitHub MCP server, or scrape competitor websites via Playwright — all without leaving the terminal.

The setup is one command: `claude mcp add <server-name> -- <command>`. Three servers is the recommended sweet spot to start.

### Key Takeaway
**Start with these 3 MCP servers for consulting workflows:**
1. **GitHub MCP** (`gh` integration) — manage PRs, issues, and code reviews without browser context-switching
2. **PostgreSQL/Supabase MCP** — let Claude query client databases directly for analysis
3. **Brave Search MCP** — give Claude web search ability for market research and competitive analysis within your sessions

Install one today with: `claude mcp add github -- npx @anthropic-ai/github-mcp-server`

## Resource 2: Custom Sub-Agents — Delegate Parallel Tasks Within a Session

**Source:** [Create custom subagents — Claude Code Docs](https://code.claude.com/docs/en/sub-agents) | [Quickstart Guide — Shipyard](https://shipyard.build/blog/claude-code-subagents-guide/)
**Time:** ~5 min read
**Category:** Workflow Optimization

### Why This Is Relevant
Sub-agents are specialized assistants that run in their own context window with custom instructions and limited tool access. For a consultant, this means you can create a `data-analyst` sub-agent that only has access to read files and run Python, and a `report-writer` sub-agent that only edits markdown files. Claude automatically delegates to them when the task matches.

The key benefit: sub-agents keep exploration and heavy analysis out of your main conversation, preserving your context window for decision-making.

### Key Takeaway
**Create your first sub-agent in 2 minutes:** Run `/agents` in Claude Code to get the interactive setup wizard. Or create a file at `.claude/agents/data-analyst.md`:

```markdown
---
name: data-analyst
description: Analyze CSV and Excel files with Python pandas
tools:
  - Read
  - Bash
  - Write
---

You are a data analyst. Use pandas to analyze data files.
Always output summary statistics, flag missing values, and
save visualizations to the figures/ directory.
```

Now when you say "analyze this dataset," Claude delegates to your custom agent with the exact constraints you defined.

## Quick Win: Add the GitHub MCP Server Right Now (1 min)

If you use GitHub for any project work, this single command eliminates the browser back-and-forth:

```bash
claude mcp add github -- npx @anthropic-ai/github-mcp-server
```

After adding it, you can do things like:
- `"Show me all open PRs on this repo"` — without opening GitHub
- `"Create an issue for the data pipeline bug we just discussed"` — Claude creates it directly
- `"Review the latest PR and leave comments"` — Claude reads the diff and comments inline

This compounds over a workweek: if you check GitHub 20+ times a day, doing it from your terminal session saves meaningful context-switching time.

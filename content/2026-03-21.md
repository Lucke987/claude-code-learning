## Quick Win — Apply This Today

**Turn your most common prompt into a slash command.**

Think of the prompt you type most often when using Claude Code. Copy it into a Markdown file at `~/.claude/commands/<name>.md`. From now on, type `/<name>` instead of re-typing the prompt.

For a BCG consultant, high-value candidates:
- `/analyze-data` — standard data analysis structure for client deliverables
- `/review-pr` — consistent PR review checklist aligned to your team's standards
- `/summarize-codebase` — structured overview for onboarding onto a new client repo

This takes 3 minutes to set up and saves 2-3 minutes every time you use it. Commands live in git, so they compound across your whole team.

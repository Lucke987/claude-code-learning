## Quick Win — Apply This Today

**Set up your first Hook in 5 minutes — a simple audit trail.**

Open `~/.claude/settings.json` and add:

```json
{
  "hooks": {
    "PostToolUse": [
      {
        "matcher": "Edit|Write",
        "hooks": [
          {
            "type": "command",
            "command": "echo \"$(date -Iseconds) | $CLAUDE_TOOL_NAME | $CLAUDE_TOOL_INPUT_FILE_PATH\" >> ~/.claude/edit-log.txt"
          }
        ]
      }
    ]
  }
}
```

This logs every file Claude touches with a timestamp — giving you a clean audit trail of what changed in each session, without relying on Claude to report it. For BCG client engagements, this is the foundation of quality control on any AI-assisted work.

**Once it's working, swap in a real linter:**
```json
"command": "cd $(dirname $CLAUDE_TOOL_INPUT_FILE_PATH) && ruff check $CLAUDE_TOOL_INPUT_FILE_PATH --fix 2>/dev/null; true"
```

Now every Python file Claude edits is automatically linted — no prompting required, every time, guaranteed.

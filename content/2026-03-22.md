## Quick Win — Apply This Today

**Set up two hooks in 5 minutes.**

Create or open `.claude/settings.json` in your current project and add:

```json
{
  "hooks": {
    "PreToolUse": [
      {
        "matcher": "Bash",
        "hooks": [
          {
            "type": "command",
            "command": "echo \"$CLAUDE_TOOL_INPUT\" | grep -qE '(rm -rf|DROP TABLE|force push|--force)' && echo 'Blocked: destructive command detected' >&2 && exit 2 || exit 0"
          }
        ]
      }
    ],
    "PostToolUse": [
      {
        "matcher": "Write|Edit",
        "hooks": [
          {
            "type": "command",
            "command": "file=$(echo \"$CLAUDE_TOOL_INPUT\" | python3 -c \"import sys,json; d=json.load(sys.stdin); print(d.get('file_path',''))\" 2>/dev/null); [ -n \"$file\" ] && prettier --write \"$file\" 2>/dev/null || true"
          }
        ]
      }
    ]
  }
}
```

That's it. Every session in this project now has a destructive-command guardrail and auto-formatting on every file write — no reminders needed. Takes 5 minutes, saves friction every day.

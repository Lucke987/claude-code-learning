## Quick Win: Apply This Today

**Set up a project CLAUDE.md in your current working repo.** Create a file called `CLAUDE.md` in the root of whatever project you're working on right now. Include:

```markdown
# Project Context
- This is a [client/internal] project for [brief description]
- Primary language: Python 3.x
- Key libraries: pandas, matplotlib, openpyxl

# Commands
- Run tests: pytest tests/
- Lint: ruff check .

# Style Rules
- Use type hints for all function signatures
- Prefer pathlib over os.path
- Output charts as both .png and .svg

# Data
- Raw data lives in data/raw/
- Processed data goes to data/processed/
- Schema: [describe your key columns/tables]
```

This takes 2 minutes and will immediately improve Claude Code's accuracy on every task in that project. You'll notice fewer corrections needed and more relevant code suggestions from the very first prompt.
